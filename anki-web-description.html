This Anki deck helps you learn key information about the 10 provinces and 3 territories of Canada. It is an improved version of the original deck Provinces and Territories of Canada (ankiweb.net/shared/info/2260936875) with additional cards covering flags, coats of arms, and postal codes for each province and territory, and a few fixes.
